Установка Python + Jupyter Notebook:

Python
1. Если у вас не установлен python3, скачайте дистрибутив с официального сайта https://www.python.org. Внимание: нужно скачивать версию 3.4, так как версия 3.5 для 32-битных систем не поддерживается. Узнать разрядность системы можно через Система->Свойства.
2. При установке нужно обязательно поставить флажок "Add python.exe to PATH", чтобы путь к исполняемым командам python и pip был записан в переменной среды.

Jupyter Notebook
1. После установки Python откройте командную строку (перезагрузите, если она была открыта)
2. Затем нужно обновить pip:
	pip install --upgrade pip
3. Чтобы установить Jupyter Notebook, запустите:
	pip install jupyter

Приложение Jupyter Notebook установлено и готово к работе.
Запуск из командной строки:
	jupyter notebook
Рабочая папка ноутбука находится в папке пользователя: C:/Пользователи(Users)/Имя_пользователя
